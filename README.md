# SmartReply
An AI powered messaging assistant that provides context-aware reply suggestions, adapts to user tone and personality, and leverages conversation history to generate, smarter, more relevant responses in real time

APIs

1. Signup a user
    a. Name
    b. username
    c. password
    d. mobile_no(optional, later)

2. Login a user
    a. username
    b. password

3. After logging in, the user will see a screen to add people to connect.
   Search user

4. Chat with other users
